C++ LIBRARY#

C/C++ Library compilation#

Here we will show how to compile a c++ library by cmake.

The first step is to use git command to clone the repository.

git clone <repository url>

To compile library, you can use the following commands:

sudo mkdir build && cd build
sudo cmake ..
sudo make install

The first command is to create a build directory and entering it. The second one is to build the library. The third command is to install the library.

Eigen library#

For installation, just run

sudo apt update
sudo apt install libeigen3-dev

Boost library#

For installation, just run

sudo apt-get update
sudo apt-get install libboost-all-dev
