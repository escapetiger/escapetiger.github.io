"useAcrylic": true,

Oh my zsh#

For detail, see Installation for Oh My Zsh.

  • When running Netch, the error message is

Connecting to raw.githubusercontent.com (raw.githubusercontent.com)|0.0.0.0|:443… failed: Connection refused.

In this case, the solution is to manually install install.sh file and run sh install.sh in command line.

  • When running SSTap, you can directly use wget or curl command.

Remark: ~/.zshrc is the config file for zsh, while ~/.bashrc is for bash. You can change the default directory when entering wsl by simply adding the following code into the right config file.

cd <default directory>

VIM#

Load config file .virmc. Download vundle plugin and run PluginInstall command to install other plugins. However, I preferred to use vim plugins on vscode, so the most important part in my config file is the mapping settings.

Gnuplot#

For installation, just run

sudo apt-get update
sudo apt-get install gnuplot

For usage, see Gnuplot documentation.

Vscode#

First , you should install vscode on windwos. Then follow the official guide to setup remote for wsl.
